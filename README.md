# 📌 Overview

This project is a fully styled **Personal Portfolio Website** created as part of my Web Development learning journey.

The main objective of this project was to master **CSS3** styling by transforming the HTML portfolio into a visually appealing, responsive website with modern design techniques like glassmorphism, animations, and flexbox/grid layouts.

---

# ✨ Features

✅ Modern Glassmorphism Design

✅ Dark Theme with Gradient Accents

✅ Responsive Navigation Bar

✅ Hero Section with Image & Text Layout

✅ Glass-styled About Section

✅ Interactive Skills with Hover Effects

✅ Project Cards with Hover Animations

✅ Styled Contact Form

# 🎨 Design System

### Color Palette

| Color Name      | Hex Code    | Usage                    |
|----------------|-------------|--------------------------|
| Background     | #000000     | Main background           |
| Glass BG       | rgba(255,255,255,0.03) | Glassmorphism cards |
| Accent Primary  | #8b5cf6     | Purple accent             |
| Accent Secondary| #a78bfa     | Light purple              |
| Text Primary   | #fafafa     | Main text                 |
| Text Secondary  | #a1a1aa     | Muted text                |

### Typography

* **Headings:** Inter (600-700 weight)
* **Body:** Inter (400-500 weight)
* **Fallback:** -apple-system, BlinkMacSystemFont, sans-serif
