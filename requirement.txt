Project Overview
----------------
- Goal: Build a personal portfolio website showcasing projects, skills, and contact information.
- Objective: Provide a clean, responsive, and easily maintainable static site using HTML, CSS, and optional JavaScript.

Setup Instructions
-------------------
1. Clone the repository:
   git clone <repository-url>
2. Open the project folder:
   cd Personal-Portfolio-Website
3. Install dependencies (if any, e.g., npm for tooling):
   npm install
4. Run a local development server (optional):
   npm run dev   # or open index.html directly in a browser
5. Deploy: copy the `dist` folder to any static hosting service (GitHub Pages, Netlify, etc.).

Code Structure
----------------
- `index.html` – Home page, introduces the developer and highlights key projects.
- `about.html` – Detailed biography and skill summary.
- `projects.html` – Grid/list of projects with screenshots and links.
- `contact.html` – Contact form (optional) and social links.
- `assets/`
  - `css/` – Stylesheets (e.g., `styles.css`, `responsive.css`).
  - `js/` – Optional JavaScript for interactivity.
  - `images/` – Profile picture, project screenshots, icons.
- `README.md` – Project documentation and usage guide.

HTML Concepts Learned
----------------------
- Semantic elements (`<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`) for improved accessibility and SEO.
- Responsive design using Flexbox and CSS Grid.
- Media queries for adapting layout to mobile, tablet, and desktop.
- Embedding multimedia (`<img>`, `<video>`, `<audio>`).
- Form handling basics (`<form>`, `<input>`, `<label>`).
- Accessibility considerations (ARIA attributes, alt text).
